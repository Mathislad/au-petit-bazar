# Au petit baz'AR — Site vitrine

Site vitrine pour **Au petit baz'AR**, commerce multiservices de village à Saint-Front (Haute-Loire).

## Stack

- [Next.js 16](https://nextjs.org) — App Router
- [Tailwind CSS v4](https://tailwindcss.com)
- [Framer Motion](https://www.framer.com/motion/)
- TypeScript
- Déployé sur [Vercel](https://vercel.com)

## Développement

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Contenu

Tout le contenu (textes, horaires, adresse, services…) est centralisé dans `lib/site-content.ts`.

## Déploiement

Le site est déployé automatiquement sur Vercel à chaque push sur `main`.

URL de production : [https://au-petit-bazar.vercel.app](https://au-petit-bazar.vercel.app)
